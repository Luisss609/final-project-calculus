Infinite Limits and Limits at Infinity

Calculus for Computer Science 20 min read

INFINITE LIMITS

Suppose f is defined for all x near a. If f(x) grows arbitrarily large for all x sufficiently close (but not equal) to a, we write:

\[ \lim_{x \to a} f(x) = \infty \]

We say the limit of f(x) as x approaches a is infinity.

If f(x) is negative and grows arbitrarily large in magnitude for all x sufficiently close (but not equal) to a, we write:

\[ \lim_{x \to a} f(x) = -\infty \]

In this case, we say the limit of f(x) as x approaches a is negative infinity. In both cases, the limit does not exist.

Derivatives: The Limit Definition

Calculus for Computer Science 25 min read

THE LIMIT DEFINITION OF DERIVATIVES

The derivative of f is the function:

\[ f'(x) = \lim_{h \to 0} \frac{f(x + h) - f(x)}{h} \]

Provided the limit exists. If f'(x) exists, we say f is differentiable at x. If f is differentiable at every point of an open interval I, we say that f is differentiable on I.

EXAMPLE

Using the limit definition of derivative, find f'(x), where f(x) = x^2.

Solution:

If f(x) = x^2, then f(x + h) = (x + h)^2.

\[ f'(x) = \lim_{h \to 0} \frac{f(x + h) - f(x)}{h} \]

\[ f'(x) = \lim_{h \to 0} \frac{(x + h)^2 - x^2}{h} \]

\[ f'(x) = \lim_{h \to 0} \frac{x^2 + 2xh + h^2 - x^2}{h} \]

\[ f'(x) = \lim_{h \to 0} \frac{2xh + h^2}{h} \]

\[ f'(x) = \lim_{h \to 0} \frac{h(2x + h)}{h} \]

\[ f'(x) = \lim_{h \to 0} 2x + h \]

\[ f'(x) = 2x + 0 \]

\[ f'(x) = 2x \]

Therefore f'(x) = 2x, using the limit definition of derivatives.

Rules of Differentiation

Calculus for Computer Science 30 min read

RULES OF DIFFERENTIATION

THEOREM 2: CONSTANT RULE

If c is a real number, then:

\[ \frac{d}{dx} (c) = 0 \]

This rule states that the derivative of a constant is always equal to zero.

Examples:

  1. \[ \frac{d}{dx} (2^8) = 0 \]
  2. \[ \frac{d}{dx} (1,000,000) = 0 \]
  3. \[ \frac{d}{dx} (40) = 0 \]
